# android-open-project-analysis
Android 开源项目源码解析，自己所在部门的一个协作项目，旨在分析优秀的开源框架，实现原理，优雅的设计，同时提高自己。

## 具体编写步骤
具体步骤请见：[编写步骤](./zen/procedure.md)
